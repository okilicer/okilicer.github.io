MNIST TensorFlow Project

Project Description

This project uses TensorFlow to classify handwritten digits from the MNIST dataset. It includes data loading, preprocessing, building a neural network, training, and evaluating accuracy. You can explore both the full PDF explanation and the Jupyter Notebook.

Skills Highlighted

  • Image Classification
  • Neural Networks with TensorFlow
  • Data Normalization and Preprocessing
  • Model Evaluation and Metrics

Key Code Snippet

import tensorflow as tf
from tensorflow.keras.datasets import mnist
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Flatten

(x_train, y_train), (x_test, y_test) = mnist.load_data()
x_train, x_test = x_train / 255.0, x_test / 255.0

model = Sequential([
    Flatten(input_shape=(28, 28)),
    Dense(128, activation='relu'),
    Dense(10, activation='softmax')
])

model.compile(optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
model.fit(x_train, y_train, epochs=5)
model.evaluate(x_test, y_test)

BERT Disaster Tweet Classification

Project Description

This project applies transfer learning with BERT (using Hugging Face Transformers) to classify disaster-related tweets. The dataset was sourced from Kaggle and includes labeled tweets. The model was fine-tuned using a pretrained BERT base, followed by evaluation on test data. The goal was to experiment with minimal compute on Kaggle GPU.

Bike Sharing Demand: GRU + Random Forest

Project Description

This project applies time-series modeling to the Kaggle Bike Sharing Demand dataset using a Bidirectional GRU neural network and ensemble methods like Random Forest and Gradient Boosting. The goal was to predict hourly bike rental counts by extracting time-based features and engineering sequences for deep learning. Early stopping and dropout regularization were also used to avoid overfitting.

Skills Highlighted

  • Time Series Forecasting
  • Bidirectional GRU with TensorFlow/Keras
  • Random Forest and Gradient Boosting (Scikit-learn)
  • Feature Engineering from Timestamps
  • Sequence Generation for RNN Input
  • Model Evaluation with RMSLE
